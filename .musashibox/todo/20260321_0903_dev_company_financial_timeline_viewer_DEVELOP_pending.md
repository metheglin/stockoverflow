# Company Financial Timeline Viewer（企業財務タイムライン表示）

## 概要

特定の企業の財務データ・指標の時系列推移をターミナル上に表形式で表示するRakeタスクを実装する。
プロジェクトの主要ユースケースである「ある企業の業績推移を確認する」を実現するための基本ツール。

## 背景

- 現在、データは蓄積されているが、特定企業のデータを俯瞰する手段がない
- プロジェクトの想定ユースケース「ある企業の業績が飛躍し始める直前にどのような変化があったかを調べる」を実現するためには、時系列データの確認が不可欠
- まずはターミナルベースの表示機能として実装し、将来のWeb UIの基盤とする

## 実装内容

### Rakeタスク

`rake company:timeline[securities_code]`

### 表示内容

証券コードを指定すると、以下の情報を時系列で表示する:

#### 1. 企業基本情報ヘッダー
- 企業名、証券コード、セクター、市場区分

#### 2. 決算推移テーブル（横軸: 年度、縦軸: 項目）

| 項目 | FY2021 | FY2022 | FY2023 | FY2024 | FY2025 |
|------|--------|--------|--------|--------|--------|
| 売上高 | ... | ... | ... | ... | ... |
| 営業利益 | ... | ... | ... | ... | ... |
| 純利益 | ... | ... | ... | ... | ... |
| EPS | ... | ... | ... | ... | ... |
| 売上高YoY | ... | ... | ... | ... | ... |
| 営業利益YoY | ... | ... | ... | ... | ... |
| 営業利益率 | ... | ... | ... | ... | ... |
| ROE | ... | ... | ... | ... | ... |
| ROA | ... | ... | ... | ... | ... |
| 営業CF | ... | ... | ... | ... | ... |
| FCF | ... | ... | ... | ... | ... |
| PER | ... | ... | ... | ... | ... |
| PBR | ... | ... | ... | ... | ... |

#### 3. 連続記録サマリー
- 連続増収: N期
- 連続増益: N期

### オプション

- `scope` 指定（consolidated / non_consolidated、デフォルト: consolidated）
- 表示年数の指定（デフォルト: 直近5期）

## 技術的注意点

- `lib/tasks/company.rake` に配置
- 数値のフォーマット（百万円単位、パーセント表示、小数点以下の桁数統一）
- ターミナル幅に応じた適切な列幅調整
- データが存在しない期は空欄で表示
- テストは表示整形のロジック部分について記述（Rakeタスク自体のテストは不要）
